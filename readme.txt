Hypho-o is a plugin that automatically can hyphenate your articles, pages and comments

== Description ==

Hypho-o is a plugin (provided by [plazoo](http://www.plazoo.com/)) that automatically hyphenates your articles, pages and comments. Hyphenation in a simple and comfortable way which fits every layout and theme.
Change your layout and your text will follow;)

Hypho-o inserts an so called softhyphen (displayed as “&amp;shy;” in the source) at all possible hyphenation points of a word. The browser displays this softhyphen as a hyphen when needed.

Hyphoo Features:

<li>Simple Hyphenation for articles, pages and comments</li>
<li>Hyphenation may be used separately for the main text or for text extracts</li>
<li>Supported languages: German, English,  Czech,  French, Dutch and Russian</li>
<li>Custom Configuration interface  for changing the default settings</li>
<li>Administration Interface in: German, Englisch, Dutch and Russian</li>
<li>No JavaScript needed</li>


== Installation ==

You can download and install "hypho-o" using the built in WordPress plugin installer. If you download "hypho-o" manually, make sure it is uploaded to "/wp-content/plugins/hyphoo/".

Activate "hypho-o" in the "Plugins" admin panel using the "Activate" link.
